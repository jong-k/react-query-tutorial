# React Query(v4) 튜토리얼
## 1. Blog-em Ipsum
### 1.1. 소개
> Lorem Ipsum 텍스트로 가상의 게시판 구현
- jsonplaceholder API에서 게시글, 작성자, 댓글 등의 데이터 수집
- React Query의 Pagination 기능을 활용해 게시판 페이지 구현
- Prefetching으로 다음 페이지 데이터를 미리 받아와 매끄러운 페이지 전환 구현
### 1.2. 구성 요소
- jsonplaceholder API
- React Query
## 2. Infinite SWAPI
> Star Wars 데이터를 나열하는 무한 스크롤 구현

### 2.2. 구성 요소
